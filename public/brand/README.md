# Incremento — Brand Pack

Two brands, one studio.
- **Incremento** — digital studio (web, product, growth)
- **Incremento Labs** — AI consultancy

## Logo Files
| File | Use |
|------|-----|
| `logo-full-dark.svg` | Primary lockup on dark backgrounds |
| `logo-full-light.svg` | Primary lockup on light backgrounds |
| `logo-labs-dark.svg` | Labs lockup on dark |
| `logo-labs-light.svg` | Labs lockup on light |
| `logo-mark.svg` | Standalone mark (transparent) |
| `logo-square-dark.svg` | App icon / social avatar (dark) |
| `logo-square-light.svg` | App icon / social avatar (light) |
| `favicon.svg` | Browser favicon |
| `favicon-32.png`, `favicon-180.png`, `favicon-512.png` | Raster favicons / Apple touch icon |

## Color Palette
| Token | Hex | Use |
|-------|-----|-----|
| Midnight | `#0F1426` | Primary background |
| Foreground | `#F4F6FA` | Primary text on dark |
| Lime | `#C7F26B` | Primary accent / gradient start |
| Cyan | `#7FE9E0` | Secondary accent / gradient end |

**Brand gradient:** `linear-gradient(135deg, #C7F26B 0%, #7FE9E0 100%)`

## Typography
- **Display / headings:** Space Grotesk (500–700)
- **Body / UI:** Inter (400–600)
- Letter-spacing: −0.02em on display

## Clear space
Maintain clear space around the mark equal to the height of the dot ("i" tittle).

## Don'ts
- Don't recolor the gradient
- Don't stretch or rotate the mark
- Don't place the dark lockup on busy photography without a scrim
